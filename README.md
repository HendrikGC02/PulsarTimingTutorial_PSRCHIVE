# PSRCHIVE tutorial — site

Three pages, plain HTML + React (loaded over CDN, no build step).

- `index.html` — landing / visual essay
- `try-it.html` — IDE-style command playground
- `reference.html` — searchable command cards

## Folders

```
site/
  index.html
  try-it.html
  reference.html
  styles/site.css
  scripts/
    diagrams.jsx       (shared header + plot mocks)
    landing.jsx
    tryit.jsx
    reference.jsx
  plots-spec.md        (handoff doc for Claude Code)
  plots/               ← generated externally; see plots-spec.md
```

## Hosting on GitHub Pages

Push the `site/` contents to the repo root (or set Pages to serve from `/site`). No build step — Babel transpiles in the browser. That's fine for a tutorial site; if it ever feels slow we can prebuild with `esbuild`.

## Theme

Light/dark stored in `localStorage` under `psrchive-theme`. The Try-it IDE keeps its dark chrome regardless — that's intentional, matches VSCode feel.

## Plot images

Not in this repo yet — see `plots-spec.md`. Real plots are generated by a separate Claude Code session against a PSRCHIVE install, then dropped into `site/plots/`.
